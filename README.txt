Aplikasi ini merupakan sebuah aplikasi web e-commerce.
Seorang user dapat memiliki 1 diantara 2 role yang berbeda:
1. administrator.
2. user (secara default).

Untuk mengakses akun admin, gunakan username dan password berikut.
  username: admin
  password: admin
Seorang admin memiliki fitur:
1. Login.
2. Mengubah data user secara manual.
3. Mengubah role user lain.
4. Menghapus user.
5. Logout

Untuk mengakses akun user biasa, gunakan username dan password berikut.
  username: wendy
  password: wendy
Seorang user biasa memiliki fitur:
1. Registrasi.
2. Login.
3. Mengubah profile.
4. Logout.
